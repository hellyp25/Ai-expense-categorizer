Expense Categoriser

Overview
This project is a basic Expense Categoriser that classifies expenses into different categories based on input data. It helps in organizing and understanding spending patterns in a simple and efficient way.

Features
* Categorizes expenses automatically
* Helps track and organize spending
* Simple and easy to use
* Suitable for basic financial analysis

Dataset
The project uses a dataset containing expense records with details such as:
* Expense description
* Amount
* Category
NOTE: A small dataset is used for demonstration purposes.

Working
* The dataset is loaded and preprocessed
* Input data is analyzed
* Expenses are classified into categories
* Results are displayed in a simple format

How to Run
```bash id="9y2kq1"
pip install -r requirements.txt
streamlit run app.py
```

Limitations
* Uses a small dataset
* Basic classification logic
* Limited categories
